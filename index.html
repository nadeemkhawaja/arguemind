System Architecture

ArgueMind Reasoning Pipeline

Multi-layer AI reasoning system using the PSIPAB framework. Each layer receives all prior layers as chained context. Telemetry is captured at every node.

PSIPAB Framework — Layer Mapping
P Problem L0 Fallacy scan + L1 Context analysis — define and map the problem space
S Solution L2 Argument builder — propose the strongest evidence-backed solution
I Implementation L3 Counter-argument — test the solution against real opposition
P Proof L4 Self-critique — audit weaknesses, validate the argument honestly
A Action L5 Final verdict — Loffi delivers definitive strategy and ruling
B Benefit Score card + coaching — measurable improvement for the debater
Telemetry Captured at Every Layer (Requirement 4)
Per layer
Input tokens · Output tokens · Latency (s) · Efficiency ratio (out÷in)
Aggregate
Total input · Total output · Total tokens · Total elapsed · Global efficiency
Optimization (Req. 5)
If eff <0.6 → recommend depth increase. If total >8000 tokens → recommend concise mode. If latency >60s → flag slow layer.
